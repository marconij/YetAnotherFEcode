# Topology Optimization Post-Processing and Validation

This directory contains scripts and tools for post-processing and validating the results of topology optimization simulations using [COMSOL Multiphysics](https://www.comsol.com/comsol-multiphysics/).

To use this toolkit, ensure you have COMSOL Multiphysics installed with the [LiveLink for MATLAB](https://www.comsol.com/livelink-for-matlab) feature enabled.

An example is provided in the `main.m` file to illustrate the usage of these tools.
